# Interview Must Do

This folder is your focused interview set.

Use this after basic practice. These questions are more important than very easy problems like reverse string, vowel count, or first letter uppercase.

## How To Practice

1. Solve the problem by yourself first.
2. Write the brute-force solution.
3. Then write the optimal solution.
4. Always note time complexity and space complexity.
5. Revise the same problem again after 2-3 days.
