An S3 bucket is a cloud storage container in AWS used to store files as objects, such as images, videos, documents, backups, and static website files.

S3 bucket ek container hota hai jisme hum files store karte hain
Jaise local system me folder hota hai 📁
waise AWS me bucket hota hai.

Why use S3 bucket?
highly scalable
secure
low cost
high availability
file hosting
