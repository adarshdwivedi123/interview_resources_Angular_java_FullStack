There are 7 common ways:

Method	Used For
Route Params	Required values (/user/101)
Query Params	Optional values (?id=101)
Navigation State	Pass objects without showing them in the URL
Shared Service (RxJS/Signals)	Component-to-component communication
@Input()	Parent → Child
@Output()	Child → Parent
Local Storage / Session Storage	Persist data across refreshes
