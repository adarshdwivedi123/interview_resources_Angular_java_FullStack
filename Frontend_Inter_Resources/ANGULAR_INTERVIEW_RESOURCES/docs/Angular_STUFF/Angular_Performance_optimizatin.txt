Angular application performance can be improved by reducing unnecessary API calls,
 and loading only required resources
 Lazy Loading Modules

1)Lazy loading 
loads modules only when required, instead of loading the entire application initially
Benefit:
{
 path: 'dashboard',
 loadChildren: () =>
 import('./dashboard/dashboard.module')
 .then(m => m.DashboardModule)
}

Benfit
1)Faster initial load
2)Smaller bundle size

2️⃣ Use OnPush Change Detection
Angular normally checks every component for changes.
Using OnPush reduces unnecessary checks.

Example:

@Component({
 changeDetection: ChangeDetectionStrategy.OnPush
})

Benefit:
Faster UI updates
Less CPU usage
