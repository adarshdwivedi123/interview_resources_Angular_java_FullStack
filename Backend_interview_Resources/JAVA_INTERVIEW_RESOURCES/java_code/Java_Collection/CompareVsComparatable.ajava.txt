#)Comparator vs Comparable


"Both Comparable and Comparator are used for sorting objects."

Differences:
1️⃣ Comparable is used for natural sorting, while Comparator is used for custom sorting.
2️⃣ Comparable uses compareTo() method, while Comparator uses compare() method.
3️⃣ Comparable modifies the class itself, while Comparator can be defined separately.

#)How do you sort elements in an ArrayList using Comparable interface?

🗣️ Answer:

Implement the Comparable interface and override the compareTo() method inside your class.

📘 Example:

class Student implements Comparable<Student> {
    int id;
    String name;

    Student(int id, String name) { this.id = id; this.name = name; }

    public int compareTo(Student s) {
        return this.name.compareTo(s.name);
    }
}

ArrayList<Student> list = new ArrayList<>();
list.add(new Student(1, "John"));
list.add(new Student(2, "Alice"));

Collections.sort(list);


📤 Output: Sorted by name.

143. How do you sort elements in an ArrayList using Comparator interface?

🗣️ Answer:

Use a Comparator when you want to define sorting outside the class.

📘 Example:

ArrayList<String> names = new ArrayList<>(Arrays.asList("Charlie", "Bob", "Alice"));
Collections.sort(names, (a, b) -> a.compareTo(b));
System.out.println(names);


✅ Or with custom class:

Collections.sort(list, (s1, s2) -> s1.id - s2.id);
