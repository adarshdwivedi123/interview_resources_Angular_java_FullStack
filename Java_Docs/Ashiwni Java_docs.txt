How to secure application using spring ?
------------------------------------------------------
	1. Security is very important for any web application
	2. To protect our application and data we need to implement security logic
	3. Spring Security concept we can use to secure our we app/rest APIs
	4. To secure our spring boot application we need to add below starter in pom.xml
<dependency>
    <groupId>org.springframework.boot</groupId>
    <artifactId>spring-boot-starter-security</artifactId>
</dependency>

Note: When we add this dependency in pom.xml then by default our app will be secured with the basic authentication. It will generate random password to access out application


How to override spring security default credentials ?
------------------------------------------------------------------------
	1. To override default credentials we can configure security user.name and user.password in properties file
