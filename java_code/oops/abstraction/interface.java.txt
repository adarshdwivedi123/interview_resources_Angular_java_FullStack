Real Life Exmaple




////

"Suppose we are building an e-commerce application. We need different payment methods like Credit Card, UPI, or Net Banking.
We can create an interface PaymentGateway which defines a method pay().
Each payment type (CreditCardPayment, UpiPayment, etc.) implements the interface and provides its own logic.
This way, our system becomes flexible and scalable, because tomorrow we can add another payment option (say PayPal) without changing the existing code."


package Interface;

interface  PaymentGateway{
    void pay(double amount);
}

class  CreditCardPayment implements  PaymentGateway{
    @Override
    public void pay(double amount) {
        System.out.println("Paid" +amount +"using credit card");
    }
}


class UpiPayment implements  PaymentGateway{

    @Override
    public void pay(double amount) {
        System.out.println("piad"+amount);
    }
}

public class PaymentInterface {
    public static void main(String[] args) {
        CreditCardPayment payment = new CreditCardPayment();
        payment.pay(1000);



        UpiPayment payment2 =new UpiPayment();
        payment2.pay(500);
    }

}

#)Can you extend an interface?
yes we can extend interface

#)Example
interface A {
    void methodA();
}

interface B extends A {
    void methodB();
}

class MyClass implements B {
    public void methodA() {
        System.out.println("Method A");
    }

    public void methodB() {
        System.out.println("Method B");
    }
}

public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.methodA();  // inherited via interface
        obj.methodB();  // defined in interface B
    }
}

 #)Can a class extend multiple interfaces?

In Java, a class cannot extend interfaces.
A class can only implement multiple interfaces using the implements keyword.

interface A {
    void methodA();
}

interface B {
    void methodB();
}

class MyClass implements A, B {   // implements, not extends
    public void methodA() {
        System.out.println("Method A");
    }

    public void methodB() {
        System.out.println("Method B");
    }
}

public class Main {
    public static void main(String[] args) {
        MyClass obj = new MyClass();
        obj.methodA();
        obj.methodB();
    }
}
