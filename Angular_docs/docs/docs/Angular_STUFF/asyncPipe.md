1)What is the async pipe?

The async pipe in Angular is used to automatically:
Subscribe to an Observable or Promise
Get the latest value
Unsubscribe automatically (very important)

---------------------------------------------------------------------------
2)Why async pipe is considered production ready
Because it:
✔ Prevents memory leaks
✔ Reduces code
✔ Handles subscription lifecycle
✔ Improves performance
✔ Works perfectly with Angular Signals
✔ Works perfectly with SSR & hydration
✔ Works with @if, @for, @def
